# Design Guide: Man Matters Shilajit Gummies — Static Ad Creatives

## Brand Overview

**Brand**: Man Matters  
**Product**: Shilajit Gummies (Active Nutrition line)  
**Target Audience**: Adult men (25–45), health-conscious, urban Indian market  
**Tone**: Clean, confident, science-backed, modern-masculine  
**Price Point**: ₹899 (MRP ₹999) — accessible premium  

---

## Visual Identity

### Logo & Brand Mark
- **Logo**: "man matters®" in lowercase, bold black sans-serif (custom wordmark)
- **Placement**: Always top-left (portrait/square) or top-right (landscape)
- **Size**: ~10–12% of canvas width
- **Color**: Black on light backgrounds; White on dark backgrounds
- **Never** rotate, distort, or recolor the logo

### Product Name Lockup
```
SHILAJIT          ← large, bold, black/white
GUMMIES           ← same size, slightly lighter weight or same
```
- Always stacked two lines, full caps
- Paired with "active nutrition" sub-label above in small caps/track-spaced text

## UI Components

### CTA Button
```
Shape: Pill / fully rounded rectangle
Background: Black (#000000)
Text: White, bold, 32–38px
Padding: 18px 40px
Width: ~60–70% of canvas
Shadow: None (flat)
```

### Price Badge
```
Old price: Light oval outline, ₹999/- with diagonal strikethrough, grey text
New price: Solid black pill, bold white text, ₹899
Gap between: 8–12px
```

### Ingredient Pill / Info Bubble (light)
```
Background: Warm cream #F5EFE0
Border: Dashed 1.5px, rounded rect
Contains: [Ingredient photo circle] + [Name bold] + [Benefit text]
```

### Floating Review Card
```
Background: White
Border-radius: 16px
Shadow: 0 4px 16px rgba(0,0,0,0.08)
Stars: 5 gold stars, ~16px
Quote: 14–16px, italic
Name: 12–14px, regular
```

### Bottom Attribute Strip
```
Format: [Attr 1]  |  [Attr 2]  |  [Attr 3]
Separator: Vertical bar |
Text: Bold or semi-bold, 22–26px
Color: Matches background inversion (black on white / white on black)
```

### Checkmark Label Row
```
✓ Label      ← Green checkmark + bold text, spaced evenly across width
Used for: test types, product claims
```

### Dashed Annotation Lines
```
Style: Dashed line (3px, gap 4px)
Color: #333 on light / #CCC on dark
Endpoint: Small filled circle or arrowhead
Used for: pointing from jar to ingredient / label
```

## Photography & Visual Props

### Product Shot — Jar
- Always show **man matters Shilajit Gummies** jar (black body, white cap)
- Can be: straight-on, slight tilt (~10–15°), floating/levitating, held by hand
- Label must be **legible** — avoid extreme angles
- Always full color, even when background is B&W

### Ingredient Props (always use real photography)
| Ingredient | Visual |
|---|---|
| Shilajit | Dark rocky mineral chunks / glossy resin smear |
| KSM-66 Ashwagandha | Dried pale beige root sticks in bowl |
| Black Musli | Dark chunky seeds/rounds in bowl |
| Chicory Root | Large beige root stick + purple chicory flower |

- Props in **glass or ceramic bowls** (clear/white)
- Flatlay or slight overhead angle
- Clean background (white, beige, dark slate)

### Gummy Visual
- Round, amber/orange-brown translucent gummies
- 2–3 gummies shown loose (not in jar)
- Used in usage/comparison templates

- Sugar cubes with red ❌/no symbol → "No Added Sugar" messaging
- Green battery icon → energy claim
- Mountain background (photographic) → aspirational/Himalayan origin
- Dumbbell / gym equipment (blurred BG) → performance market

## Do's and Don'ts

### ✅ Do
- Use real ingredient photography (no icons or illustrations for ingredients)
- Keep layout uncluttered — maximum 1 hero message per static
- Use dashed annotation lines for info-graphic-style layouts
- Maintain consistent jar presentation across all statics
- Use the warm beige palette for ingredient/lifestyle ads; cool grey/white for clinical/proof ads
- Keep CTA text action-oriented: "Shop Now", "Download Now"
- Use strikethrough price (₹999 → ₹899) only in price-focused statics

### ❌ Don't
- Don't use purple, neon, or unrelated brand colors
- Don't use stock photo humans (unless it's a lifestyle shot of a real Indian man)
- Don't put more than 3 claims in a single static
- Don't mix warm and cool backgrounds in one layout
- Don't use the "Download Now" CTA in organic/brand-awareness statics (only in paid/app-download ads)
- Don't show the gummies looking artificial or artificially colored
- Don't use serif fonts for any headline — brand is sans-serif throughout
